# Voting Similarity Network

Select a chamber and Congress to see how similarly members voted across all
roll-call votes in that term. Each node is a member, colored by party. An edge
means two members cast the same vote (Yea or Nay) at least as often as the
agreement threshold — drag the slider to show more or fewer connections.

The force layout pulls members with high agreement closer together, so party
clusters emerge naturally. Drag any node to reposition it, scroll to zoom.

**Note:** The House has ~1,200 votes and 435 members per Congress, so loading
takes 10–20 seconds. The Senate (~700 votes, 100 members) loads in a few seconds.

Data is queried live from the [GovQL GraphQL API](https://api.govql.us/graphql).
